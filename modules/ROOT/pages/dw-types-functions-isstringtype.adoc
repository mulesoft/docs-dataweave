[[isstringtype1]]
== isStringType&#40;t: Type&#41;: Boolean

Returns `true` if the input is the String type.

_Introduced in DataWeave: 2.3.0._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| t | The type to check.
|===

=== Example

This example shows how `isStringType` behaves with different inputs.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import * from dw::core::Types
type AString = String
output application/json
---
{
   a: isStringType(AString),
   b: isStringType(String),
   c: isStringType(Boolean),
}
----

==== Output

[source,Json,linenums]
----
{
  "a": true,
  "b": true,
  "c": false
}
----
