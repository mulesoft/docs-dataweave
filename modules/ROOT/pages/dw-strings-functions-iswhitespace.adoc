[[iswhitespace1]]
== isWhitespace&#40;text: String&#41;: Boolean

Checks if the `text` contains only whitespace.


_Introduced in DataWeave 2.2.0. Supported by Mule 4.2 and later._

_Introduced in DataWeave: 2.2.0._

=== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `text` | The input string.
|===

=== Example

This example shows how `isWhitespace` behaves with different inputs and sizes.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import isWhitespace from dw::core::Strings
output application/json
---
{
  "a": isWhitespace(null),
  "b": isWhitespace(""),
  "c": isWhitespace("  "),
  "d": isWhitespace("abc"),
  "e": isWhitespace("ab2c"),
  "f": isWhitespace("ab-c")
}
----

==== Output

[source,JSON,linenums]
----
{
  "a": false,
  "b": true,
  "c": true,
  "d": false,
  "e": false,
  "f": false
}
----
