[[dataformatsdescriptor1]]
== dataFormatsDescriptor&#40;&#41;: Array<DataFormatDescriptor&#62;

Returns the description of all the DataFormats that are installed in the current Runtime

_Experimental_: This is an experimental feature that is subject to change or removal from future versions of DataWeave.

=== Example

This example shows how the `dataFormatsDescriptor` behaves under different inputs.

==== Source

[source,DataWeave,linenums]
----
import * from dw::Runtime
---
dataFormatsDescriptor()
----
