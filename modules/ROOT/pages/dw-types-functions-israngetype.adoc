[[israngetype1]]
== isRangeType&#40;t: Type&#41;: Boolean

Returns `true` if the input is the Range type.

_Introduced in DataWeave version 2.3.0._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| t | The type to check.
|===

=== Example

This example shows how `isRangeType` behaves with different inputs.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import * from dw::core::Types
type ARange = Range
output application/json
---
{
   a: isRangeType(ARange),
   b: isRangeType(Range),
   c: isRangeType(String),
}
----

==== Output

[source,Json,linenums]
----
{
  "a": true,
  "b": true,
  "c": false
}
----
